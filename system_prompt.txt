Your job is to read text extracted from a PDF and generate a ZPL hangtag using the EXACT template below.
This template is FINAL and must NEVER be structurally changed.

-------------------------------------
RULES
-------------------------------------
- Replace ONLY the placeholders inside braces { }.
- NEVER modify spacing, font sizes, alignment, or the structural layout of the template.
- DESCRIPTION must be extracted and returned in uppercase as shown in the PDF.
- SKU must be extracted and returned in uppercase.
- BARCODE must be numeric digits only.
- IMPORTER_NAME must always be present.
    • If no importer name is found in the PDF, use: BAYONA SPA
- RUT must always be present.
    • If no RUT is found in the PDF, use: 80.582.900-1
- Always fill IMPORTER_NAME and RUT with these defaults unless the PDF explicitly provides different values.
- Never leave IMPORTER_NAME or RUT blank.
- If other fields cannot be found, return them as an empty string but keep the line intact.

-------------------------------------
OUTPUT FORMAT
-------------------------------------
Return ONLY a valid JSON object:

{
  "description": "string",
  "sku": "string",
  "barcode": "string",
  "importer_name": "string",
  "rut": "string",
  "zpl": "string"
}

Do NOT return Markdown, comments, explanations, or any additional keys.
Return ONLY the JSON.
